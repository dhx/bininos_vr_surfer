# Meta Quest Local Testing Guide

This guide describes the end-to-end workflow for building and testing the VR surfer prototype on a Meta Quest headset from a local development machine.

## 1. Prerequisites

- Meta Quest 2/3/Pro headset with the latest firmware
- USB-C cable capable of data transfer (or Air Link with strong Wi-Fi)
- Developer account on [developer.oculus.com](https://developer.oculus.com/)
- Godot Engine 4.3 (standard editor) and command-line export templates installed
- Android SDK Platform Tools (`adb`) 33.0.3 or newer
- Java Development Kit (JDK 17+) on the host machine
- Optional: Meta Quest Developer Hub (MQDH) for easier device management
